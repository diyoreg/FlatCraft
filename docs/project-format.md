# Формат проекта FlatCraft v2 (project.json)

Один проект = один JSON-файл (UTF-8), например `projects/<имя>/project.json`.
Все размеры — **миллиметры**. Цвета — HEX (`#RRGGBB`).

Этот же файл — общий язык между веб-редактором (2D) и сборщиком в 3ds Max (3D).

## Структура

```jsonc
{
  "formatVersion": 2,
  "name": "Имя проекта",
  "units": "mm",

  "room": {
    "length": 4200,        // размер комнаты по X (вдоль южной стены)
    "width": 3000,         // размер по Y
    "height": 2700,        // высота потолка
    "wallThickness": 400,
    "ceiling": false,      // строить ли потолок (false — удобнее смотреть сверху)
    "noShell": false       // true — «Убрать стены и пол»: коробка комнаты
                           // (пол, стены, потолок) не строится совсем
  },

  "kitchen": {
    "wall": "south",       // пока поддерживается только южная стена (y=0)
    "offsetX": 0,          // отступ первого блока от левого угла

    "dims": { ... },       // стандартные размеры, см. таблицу ниже (всё опционально)

    "lower": {             // нижний ряд: блоки слева направо
      "blocks": [
        { "type": "cabinet", "width": 600 },                  // шкаф (полный корпус)
        { "type": "drawers", "width": 600, "drawers": 3 },     // тумба с ящиками
        { "type": "tall",    "width": 600, "label": "пенал" }, // пенал на всю высоту
        { "type": "gap",     "width": 600, "tall": false, "label": "плита" }
      ]
    },

    "upper": {             // верхний ряд
      "enabled": true,
      "offsetX": 0,        // доп. отступ верхнего ряда относительно нижнего
      "blocks": [
        { "type": "cabinet", "width": 600 },
        { "type": "gap", "width": 600, "label": "вытяжка" }    // пустой слот
      ],
      "antresol": false    // антресоль одним блоком над всем верхним рядом
    },

    "colors": {            // все опциональны, есть значения по умолчанию
      "walls": "#EDEDED", "floor": "#B9975B",
      "lower": "#3A3F45", "upper": "#E9E4DA",
      "countertop": "#23262B", "plinth": "#1C1E22", "backsplash": "#CFC8BB"
    }
  }
}
```

## Типы блоков

| type      | Где        | Что строится                                              |
|-----------|------------|-----------------------------------------------------------|
| `cabinet` | низ / верх | Полный корпус (бокс) нужной ширины                        |
| `drawers` | низ        | N горизонтальных коробов с зазорами (`drawers`: число)    |
| `tall`    | низ        | Пенал от цоколя до верха верхних шкафов                   |
| `gap`     | низ / верх | Пустое место под технику. `label` — подпись (для 2D).     |

Свойство `tall` у нижнего `gap`:
- `false` — техника под столешницу (плита, посудомойка): столешница, фартук
  и цоколь **проходят** над/под проёмом без разрыва;
- `true` — отдельно стоящая техника (холодильник): столешница, фартук и цоколь
  **разрываются** на ширину проёма.

## Автоматика сборщика

- Между всеми блоками автоматически вставляется зазор `dims.gap` (по умолчанию 4 мм).
- Теневой зазор (`shadowGap`) — реальная пустота между верхом нижних корпусов
  и низом столешницы, геометрия для него не строится.
- Столешница строится сегментами по непрерывным участкам нижнего ряда
  (с вылетом `countertopOverhang`), фартук — над теми же сегментами.
- Цоколь — сплошной блок от стены до линии `lowerDepth - plinthRecess`
  (по умолчанию 590 мм: фронт корпусов минус 10 мм будущего фасада).

## Стандартные размеры (dims, значения по умолчанию)

| Ключ | Умолч. | Описание |
|------|--------|----------|
| plinthHeight | 100 | Высота цоколя |
| plinthRecess | 10 | Отступ цоколя от фронта (= толщина будущих фасадов: цоколь 590 при глубине 600) |
| lowerHeight | 720 | Высота нижних корпусов |
| lowerDepth | 600 | Глубина нижних корпусов |
| gap | 4 | Зазор между блоками |
| shadowGap | 4 | Теневой зазор под столешницей |
| countertopThickness | 40 | Толщина столешницы |
| countertopOverhang | 40 | Вылет столешницы |
| backsplashGap | 600 | Высота фартука (столешница → верхние) |
| backsplashThickness | 12 | Толщина панели фартука |
| upperHeight | 720 | Высота верхних шкафов |
| upperDepth | 350 | Глубина верхних шкафов |
| antresolHeight | 360 | Высота антресоли |

## Высотная схема (при размерах по умолчанию)

```
Z=0      пол
Z=100    верх цоколя / низ нижних корпусов
Z=820    верх нижних корпусов
Z=824    низ столешницы (теневой зазор 4)
Z=864    верх столешницы
Z=1464   низ верхних шкафов (фартук 600)
Z=2184   верх верхних шкафов
Z=2700   потолок
```

## Пока не поддерживается (план)

- Проёмы в стенах (окна/двери) — вернём позже.
- Стены кроме южной, L/U-раскладки, остров.
- Текстуры, стили фасадов, ручки.
